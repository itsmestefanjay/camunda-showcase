## Camunda Showcase
This show case gets into a bit more advanced techniques in Camunda BPM:

- Error handling
- Signals
- Messages
- Compensation
- and more

It will be filled with more examples over time

## Prerequisits
Basic knowledge in
- BPMN 2.0
- Camunda BPM (v7.12+)
- SpringBoot (2.2+)
- Java (8+)
- Gradle (6+)
